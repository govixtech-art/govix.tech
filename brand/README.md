# Govix Tech — Brand System

Este diretório contém os arquivos oficiais de identidade da Govix Tech.

## Uso obrigatório
- Todos os sites
- Todos os SaaS
- Materiais institucionais

## Proibido
- Criar cores fora dos tokens
- Usar fontes diferentes
- Alterar logotipo
- Inserir efeitos visuais

## Princípio
A marca Govix Tech é discreta, técnica e confiável.
Ela sustenta o conteúdo. Não compete com ele.
